# Introduction

This repository contains a base repository leveraging the following:

1. Typescript
1. Jest for testing

A Sample module `src/sample.ts` has been configured along with it's associated test `__tests__/sample.test.ts`. 

## Getting started

```
npm install
```

## Running Tests

```
npm t
```
